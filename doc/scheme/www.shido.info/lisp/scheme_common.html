Appendix 5. 記事中のコードを実行する方法

この解説記事では R5RS で定義されていないいくつかの関数を使っているので、実行するためには、 本文中で明記しているもの以外に、 以下のコードをあらかじめロードする必要があります。
（他にもあったらお知らせください。） 

(define call/cc call-with-current-continuation)

(define (inc x)
  (+ x 1))

(define (dec x)
  (- x 1))
