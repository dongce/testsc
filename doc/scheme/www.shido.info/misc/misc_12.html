自宅サーバを立ち上げた理由

Feb 09, 2006

自宅サーバーを立ち上げた理由は以下の４つです。

  1. メールをどの PC でも読めるように IMAP を使いたかったから。
  2. 動的な web ページを作りたかったから。
  3. 電気代以外は余計な出費がないから。
  4. プロバイダーを乗り換えてもドメインネームが変わらないから。 (DynDNS がつぶれなければ...)
新しいホームページも見た目は1つも動的ではないのですが、とりあえず、 以下のことができるようになっています。
  1. トップページの花の挿絵を自動で入れ替える。
  2. 更新ログをデータベース化し、トップページと更新ログが 自動で更新されるようにする。
今後以下のことを行う予定です。
  1. 備忘録をデータベース化する。
  2. ゲストブックを作る。

引越しに伴いご面倒をおかけしますが、より良いページを作るよう努力する所存ですので、 よろしくご理解のほどをお願いします。
