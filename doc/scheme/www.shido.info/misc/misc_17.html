YBB 8M は固定 IP アドレス？

Jul 09, 2006

自宅サーバを立ち上げて半年近くたつのに、一度も IP address は変化しませんでした。 この間、ADLS モデムの電源を落としたことはしばしばありました。

紫藤は初期の Yahoo BB 8 M に加入し、そのまま使っています。 このサービスは公式には dynamic IP address ですが、実は固定 IP address サービス なのではないでしょうか？月額 2500 円で固定 IP が手に入ってとてもラッキーです。 光に乗り換えるのは見合わせて、しばらくこのサービスを利用しようと思っています。 訪問者の数から見積もって、上り 1 Mbps でまったく問題ないみたいです。 shidot.dyndns.org は DynDNS のダイナミック IP address サービスを利用していたのですが、 同社の固定 IP address サービスに変更しました(ともに無料）。

この変更に伴い一時 shidot.dyndns.org では紫藤のホームページにつながりなくなりました。 事前の連絡無しに不通にしたことをお詫び申し上げます。
