Linux (GNU Debian) へのソフトのインストール

Jun 28, 2005

Linux (GNU Debian) にソフトをインストール方法を簡単に述べたいと思います。 Debian のインストールの方法などは、 Debian JP Projectを見てください。

1. バイナリーをインストール

  1. バイナリーがあるかどうかを apt-cache search で調べます。 例えば、python で SQL 関連のモジュールを調べるには以下のようにします。 
    >apt-cache search python | grep -i SQL | w3m
    詳しくは Debian GNU/Linux 使用方法 簡易マニュアル: 5.1. 目的のパッケージを検索するには を見てください。
  2. モジュールが見つかったら root になって apt-get install します。 詳しくは Debian GNU/Linux 使用方法 簡易マニュアル:5.2. パッケージのインストール、削除 を見てください。

2. ソースからインストール

  1. ソースをダウンロードしてどこかのディレクトリに展開します。
  2. configure を実行します。
  3. configure にモジュールが見当たらないと云われたらそれをインストールします。 多くの場合バイナリーを apt-get できますから、モジュール名の終わりに "-dev"がついたものをインストールして、 configure をやり直します。
  4. configure が通ったら、make, make install します。
