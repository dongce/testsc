 xyzzy は亀井さんが開発した emacs 風のエディターで、win32 上で動作します。
emacs より動作が軽快で OS との相性も良いです。
紫藤が使い始めたときは（一部の熱狂的なユーザーはいたものの）
かなりマイナーなエディターで周りの人に奇異の目で見られたものですが、
現在は認知度が上がり、教育機関でも使われているし、 本も出版されました。
誠に喜ばしい限りです。
